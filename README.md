# Salinas App Store — Umbrel Community App Store

Apps personalizadas para Umbrel.

## Cómo añadir esta tienda a tu Umbrel

1. Ve a **App Store** en tu Umbrel
2. Haz clic en los **tres puntos** (⋯) arriba a la derecha
3. Selecciona **"Add community app store"**
4. Pega esta URL:
   ```
   https://github.com/Salinas25lz/umbrel-app-store
   ```
5. Confirma — la tienda aparecerá en el App Store
